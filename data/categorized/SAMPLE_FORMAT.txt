This is a sample file showing the expected CSV format for pre-categorized transaction files.

Your CSV files in this directory should have these columns:

Date,Description,Amount,Appears On Your Statement As,Category
2023-01-15,AMAZON.COM,25.99,AMZN Mktp US,Shopping
2023-01-16,GROCERY STORE,52.47,SAFEWAY #1234,Groceries
2023-01-17,NETFLIX.COM,-13.99,NETFLIX.COM,Entertainment

Notes:
- Date format should be YYYY-MM-DD
- Amount is a numeric value (negative for credits)
- Category should be a consistent naming scheme across all your files
- These pre-categorized files are used to train the model
